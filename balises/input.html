INPUT

Définitions:

La balise INPUT désigne une zone d'entrée où un utilisateur peut entrer des données
Il est utilisé dans un élément

<form> en indiquant de quelle manière l'utilisateur peut entrer des données
Les données que l'on ajoute peuvent être divers.

Exemple:

<form action="/action_page.php"> First name: <input type="text" name="fname"><br> Last name: <input type="text" name="lname"><br> <input type="submit" value="Submit"> </form>
